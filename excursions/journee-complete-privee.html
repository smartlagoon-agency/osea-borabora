Journée Complète Privée — Le lagon comme si c'était le vôtre

7h30 Max 4 pers. Programme sur mesure
Demander un devis

7h30 en exclusivité dans le lagon de Bora Bora

La journée complète privée, c'est le tour O-sea le plus premium. 7h30 avec Hiro pour vous seuls — snorkeling le matin dans les spots que vous choisissez avec lui, déjeuner local sur un motu l'après-midi, temps libre face au Mont Otemanu. Maximum 4 personnes.

Hiro construit l'itinéraire avec vous avant de partir. Si vous avez des enfants, il adapte les spots à leur niveau. Si vous êtes photographes, il connaît les heures où la lumière sous-marine est parfaite. Si vous avez vu des raies manta la veille, il ira dans la passe où elles passent en matinée.

C'est aussi le tour recommandé par Hiro pour les familles qui veulent combiner toutes les expériences du lagon en une seule journée sans courir — à leur rythme, dans des zones adaptées, avec une attention individuelle impossible dans un groupe de 10.

  • ✓ Briefing marin personnalisé
  • ✓ Équipement snorkeling complet
  • ✓ Gilets de sécurité
  • ✓ Boissons fraîches toute la journée
  • ✓ Snacks locaux
  • ✓ Déjeuner local sur le motu
  • ✓ Serviettes
  • ✓ Programme entièrement adapté
