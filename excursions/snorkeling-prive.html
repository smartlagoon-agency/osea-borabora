Snorkeling privé — votre programme, votre rythme

Quand vous réservez le snorkeling privé, vous avez Hiro pour vous seuls — et le lagon de Bora Bora avec. Maximum 4 personnes. Pas de compromis sur les spots, pas d'attente pour les autres, pas de briefing collectif pressé.

Hiro construit l'itinéraire avec vous avant de partir. Si vous êtes passionnés de photographie sous-marine, il ira dans les zones où la lumière est la meilleure. Si vous avez un enfant peu confiant dans l'eau, il choisira des zones calmes et peu profondes. Si vous voulez voir les requins citron dans les passes, il connaît leurs horaires.

Ce tour est particulièrement recommandé pour les couples en lune de miel, les familles avec de jeunes enfants, ou toute personne qui veut une expérience vraiment personnalisée — pas une sortie standardisée.
